# ESLint & Prettier Setup

This project uses the Airbnb style guide for ESLint and Prettier for code formatting.

## Usage

- Run `npx eslint .` to lint all JavaScript files.
- Run `npx prettier --write .` to format all files.

## Integration

- ESLint and Prettier configs are in `.eslintrc.json` and `.prettierrc`.
- Ignore files: `.eslintignore`, `.prettierignore`.
- Recommended: Install VS Code extensions for ESLint and Prettier.

## Customization

Edit the config files to adjust rules or formatting preferences.
